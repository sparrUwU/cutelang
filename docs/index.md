# Welcome to CuteLang (´｡• ᵕ •｡`)

**CuteLang** is a purely functional programming language designed to be as adorable as it is powerful. Every token is a kaomoji, every expression is lambda calculus, every program is a work of art!

## Why CuteLang?

- **100% Functional** — Based on lambda calculus, with closures, currying, and first-class functions
- **Turing Complete** — Recursion, conditionals, and everything you need for real computation
- **Lazy Evaluation** — Don't compute what you don't need! Just put a sleepy face on it `(z_Z)`
- **Adorable Syntax** — Conditional? That's `(?_?)`! Happy path? `(T_T)`! Something went wrong? `(E_E)`
