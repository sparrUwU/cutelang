## 📋 О проекте

**CuteLang:3** — это эзотерический функциональный язык программирования, разработанный
в рамках курсового проекта по функциональному программированию. Язык основан на
лямбда-исчислении и реализован на **F#** с использованием платформы **.NET 8.0**.

### Концепция

Каждая синтаксическая конструкция языка — это **каомодзи** (японский текстовый смайлик).
Код на CuteLang выглядит как эмоциональная переписка, но под капотом — строгое
лямбда-исчисление с замыканиями, каррированием и ленивыми вычислениями.

## ✨ Реализованные возможности языка

| Возможность | Статус | Синтаксис | Кто сделал |
|-------------|--------|-----------|------------|
| Именованные привязки (let) | ✅ Готово | `(^_^)~` | Кира |
| Рекурсия | ✅ Готово | — | Кира |
| Ленивые вычисления | ✅ Готово | `(z_Z)` / `(O_O)!` | Кира |
| Функции первого класса | ✅ Готово | `((F_F)>>` | Кирилл |
| Замыкания | ✅ Готово | — | Кира |
| Списки / Последовательности | ✅ Готово | `>V<`, `head`, `tail`, `[]` | Кирилл |
| Библиотечные функции списков | ✅ Готово | `head`, `tail`, `>V<` | Кирилл |
| Ввод-вывод | ✅ Готово | `(^o^)>`, `<(^o^)` | Кира |
| Арифметические операции | ✅ Готово | `:+`, `:-`, `:*`, `:/`, `:%` | Кирилл |
| Операции сравнения | ✅ Готово | `B=`, `<B=`, `>B=`, `<B`, `>B`, `!B=` | Кирилл |
